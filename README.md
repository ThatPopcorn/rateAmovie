# rateAmovie

A small Flask app for rating movies.

## Overview

This repository contains a lightweight Flask application to let users rate and review movies. It includes a minimal app structure under the `app/` package.

## Requirements

- Python 3.10+
- Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

Start the app with:

```bash
python run.py
```

Then open http://127.0.0.1:5000/ in your browser.

## Project structure

- `app/` - application package (routes, models, auth, config, extensions)
- `run.py` - entry point to run the app
- `requirements.txt` - Python dependencies
- `instance/` - instance-specific configuration
- `LICENSE` - project license
